WPdo

An easier way to contribute to WordPress

About

Contributing to WordPress can be hard to start with, and even experienced developer can appreciate a simpler tool to automate their work. WPdo is a simple script that helps setting up and interacting with the tickets from Make WordPress.

Installation

  1. Open Terminal
  2. Type: curl -s https://raw.githubusercontent.com/folletto/WPdo/master/wpdo -o wpdo.php && php wpdo.php install
  3. Done!
